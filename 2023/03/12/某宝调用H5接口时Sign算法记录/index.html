2023/3/12

其中有一个Sign算法

sign=ccba0f316e36364b1c0c9661625951f9

从Js打断点跟进去 走到请求，查看生成的参数名称

K.MD5(“appid=” + X + “&business_site=” + Y + “&version=” + Z.ver + “&dimensions=” + ia + “&extend_param=” + ra),

Fa = K.MD5(“appid=” + X + “&business_site=” + Y + “&version=” + Z.ver + “&dimensions=” + ia + “&extend_param=” + ra)

根据不同的接口，可以跟到不同的加密点，原理是一样的，没什么技术含量

将参数拼接以后 进行MD5 得到Sign

接口调用频繁会出现滑块认证 滑块已经解决X82Y 这边就不放出来了。
